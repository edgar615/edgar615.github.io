2. 将线程PID转化为16进制

```
# printf 0x%x 4450
0x1162
```

3. jstack查看堆栈

```
jstack 4432 | grep 0x1162
jstack 4432 | grep -A 10  0x1162 #只看堆栈后面10行
```
